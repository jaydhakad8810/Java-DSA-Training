## ▶️ How to Run

Clone the repository:

```bash
git clone https://github.com/jaydhakad8810/Java-DSA-Training.git
```

Navigate to the project:

```bash
cd Java-DSA-Training
```

Compile Java files:

```bash
javac FileName.java
```

Run Java files:

```bash
java FileName
```
